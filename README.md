<h2 id="features">Features</h2>

* 实现照片展现、存储
* 实现简单的用户系统
* 制定打分评价系统
* 展现照片排名
* 记录用户操作日志

<h2 id="installation">Installation</h2>

*   CMake, Qt REQUIRED

* How to compile a cmake project

    ```
    cd path/to/facemash2
    mkdir build
    cd build
    cmake .. -DCMAKE_PREFIX_PATH=path/to/qt/5.x
    make
    ```

* You will find the executables under the folder bin
